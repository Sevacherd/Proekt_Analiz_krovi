Программа создана,чтобы помочь в расшифровке общего анализа крови
!!!ВНИМАНИЕ!!!
-Внимательно вносите показатели,для удобства они указанны в английской абривиатуре.
-Необязательно вносить все показатели,программа считает только внесенные Вами и выдаст результат.
-Выбирая показатель вводится только его число, без его измерений ( таких как г/л,% и т.д).
-Если показатель ввиде нецелого числа - вводится через точку(Например 1.2)
-После внесения всех интересующих показателей и убедившись в правильности нажмите кнопку "Расщифровать анализ"
-В папке с файлом(программой) будет создан файл rezult.txt c расшифровкой анализа
